# Vulnerable-Management-Program-Implementalion

# Technology Used

- Tenable – Enterprise vulnerability management and scanning
- Microsoft Azure – Hosted virtual machines for the scan engine and targets (Nessus scan engine + scan targets)
- PowerShell & Bash – Automated scripts for patching and remediation


In this project, i simulate the implementation of a comprehensive vulnerability management program, from inception to completion.

_**Inception State:**_ the organization has no existing policy or vulnerability management practices in place.

_**Completion State:**_ a formal policy is enacted, stakeholder buy-in is secured, and a full cycle of organization-wide vulnerability remediation is successfully completed.
